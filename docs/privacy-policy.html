Privacy Policy

Aun Postman · Last updated: April 5, 2026

In short: Aun Postman does not collect, store, or share any personal data. Everything stays on your device.

About Aun Postman

Aun Postman is an API testing and HTTP client application for iOS, developed by Maulik Raja. It allows you to build and send HTTP requests, test WebSocket connections, manage request collections, and inspect API responses — all from your iPhone or iPad.

Data We Collect

We do not collect any personal data. Aun Postman is a fully offline tool. All data you create within the app — including your API requests, collections, headers, authentication credentials, request history, and responses — is stored locally on your device.

Data Storage

Your data is stored in the following ways:

  • Local device storage: All app data (collections, history, settings) is stored locally on your device using encrypted local storage.
  • iOS Keychain: Sensitive information such as authentication tokens and credentials are stored securely in the iOS Keychain, which is protected by your device's hardware encryption.
  • iCloud Documents (optional): If you choose to use the iCloud backup feature, a copy of your collections and settings is saved to your personal iCloud Documents container. This data is encrypted by Apple and synced only to your own iCloud account. We have no access to this data.

Device Permissions

Aun Postman may request the following permissions:

  • Photo Library (read): Used only when you choose to attach a file or image to an API request body. We never access your photos without your explicit action.
  • Photo Library (write): Used only when you choose to save an exported file or response to your photo library.
  • Local Notifications: Used to alert you when long-running API requests complete. These are local notifications only — no data is sent to any server.

Network Activity

Aun Postman sends HTTP/HTTPS requests and establishes WebSocket connections only to the URLs that you explicitly configure. The app does not contact any analytics services, advertising networks, or telemetry servers. The app requires unrestricted network access (NSAllowsArbitraryLoads) because, as an API testing tool, it must be able to reach any server you specify, including non-HTTPS endpoints.

Third-Party Services

Aun Postman does not integrate any third-party analytics, advertising, crash reporting, or tracking services. No data is shared with any third party.

Data Sharing

We do not sell, trade, rent, or otherwise share any user data with anyone. Since we don't collect data in the first place, there is nothing to share.

Children's Privacy

Aun Postman is a developer tool and is not directed at children under 13. We do not knowingly collect any information from children.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date. Continued use of the app after changes are posted constitutes acceptance of the revised policy.
